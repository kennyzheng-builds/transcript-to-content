# Style Profiles

This directory stores persistent writing style profiles. Once saved, the skill auto-loads the default profile on every run — no need to re-upload reference articles.

## How It Works

- `default.md` — Your primary style profile, auto-loaded every time
- Create additional profiles (e.g., `formal.md`, `casual.md`) and specify by name when needed

## Creating a Profile

Tell Claude: "帮我保存风格" or "save my style profile" and provide 2-5 reference posts. Claude will analyze and save the profile here.

## Updating a Profile

Tell Claude: "更新我的风格" or "update my style" with new reference posts.

## Profile Format

Each profile captures:
- Information density pattern
- Language style and tone
- Structure conventions (headings, lists, paragraph length, opening style)
- Attitude markers
- Platform conventions
- Specific formatting habits
